# WeatherGo

WeatherGo is a mobile application developed with React Native. It provides users with weather forecasts for the next 7 days.

## Covered Topics

- React Native
- Navigation
- Weather API
- Responsiveness
- TailwindCSS (Nativewind)
- Cross-Platform Compatibility
